# Personal Workspace

This directory is for your personal working files that don't need to be shared.

**Contents are gitignored** - only this README is tracked.

## Suggested Structure

```
personal/
├── README.md          # This file (tracked)
├── notes/             # Your working notes
├── drafts/            # Draft documents
└── analysis/          # Investigation outputs
```

## Usage

- Store issue-specific analysis and notes here
- Files won't clutter the shared repo
- Reference files in GitHub issues if others need to see them
